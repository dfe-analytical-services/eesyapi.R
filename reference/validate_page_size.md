# Validate page size

Validate page size

## Usage

``` r
validate_page_size(page_size, min = 1, max = 40)
```

## Arguments

- page_size:

  Chosen page size

- min:

  Minimum valid page_size for EES API

- max:

  Maximum valid page_size for EES API

## Value

Logic

## Examples

``` r
eesyapi:::validate_page_size(20)
```
