# Look-up for API geographic_level shorthands

A look-up of API geographic_level shorthands and their respective
natural forms.

## Usage

``` r
geog_level_lookup
```

## Format

### `geog_level_lookup`

A data frame with 18 rows and 2 columns:

- api_friendly:

  API geographic level shorthands

- human_friendly:

  Natural language geographic levels
